# PIXXA - Pizza Delivery App

PIXXA is an Android pizza delivery application developed using Java and Android Studio. It allows users to browse pizzas, add items to their cart, manage delivery addresses, and place orders through a simple and user-friendly interface.

## Features

- User Registration and Login
- Browse Pizza Menu
- Add to Cart
- Increase and Decrease Item Quantity
- Automatic Price Calculation
- Delivery Address Management
- Place Orders
- Order History
- User Profile
- Material Design Interface

## Technologies Used

- Java
- Android Studio
- SQLite
- XML
- RecyclerView
- SharedPreferences
- Material Components

## Installation

1. Clone the repository:

```bash
git clone https://github.com/tejasbajaj01/Pizza-Delivery-App.git
```

2. Open the project in Android Studio.

3. Sync the Gradle files.

4. Run the application on an Android device or emulator.
